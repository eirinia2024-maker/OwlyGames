Как добавить свои игры

Откройте этот файл и отредактируйте только JSON-массив GAMES (внизу). Добавляйте записи с title, url, tags и (по желанию) thumb.

  • url — ссылка на игру (HTML-страница).
  • thumb — ссылка на картинку-превью (можно оставить пустой: будет градиент).
  • Чтобы создать новую карусель — измените/добавьте поле section (уникальные значения дадут секции).
